Inteligência de Localização de Guide Shops

Projeto desenvolvido usando a Linguagem Golang, a API do Google Maps, MySQL e dados de geolocalização para criar uma inteligência que transformasse os dados em insumos para a estratégia da empresa na qual faço estágio. Através da leitura da tabela que criei foi feito um dashboard no estilo Hitmap (mapa de calor) para facilitar a vizualização.

Conquistei o primeiro lugar da empresa no ano de 2022 por unir a parte técnica com a parte de negócios e por ter criado próximos passos relevantes para a ferramenta desenvolvida.

Aulas lúdicas de Francês
